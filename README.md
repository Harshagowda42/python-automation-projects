\# Python Automation Projects



This repository contains beginner-to-intermediate Python automation projects focused on system monitoring, log analysis, APIs, and deployment automation.



\---



\# Projects Included



\## 1. Server Health Checker



A Python automation script that monitors:



\- CPU Usage

\- Memory Usage

\- Disk Usage



\### Technologies Used

\- Python

\- psutil



\### Features

\- Real-time system monitoring

\- CPU usage alerts

\- Lightweight automation script



\### Run



```bash

python health\_checker.py

```



\---



\## 2. Log Analyzer



A Python script that analyzes server logs and detects:



\- Errors

\- Warnings

\- Failed operations



\### Technologies Used

\- Python

\- File Handling



\### Features

\- Reads log files

\- Detects ERROR messages

\- Counts total errors



\### Run



```bash

python log\_analyzer.py

```



\---



\## 3. API Testing Project



A Python project that works with APIs using the Requests library.



\### Technologies Used

\- Python

\- Requests

\- JSON



\### Features

\- Sends API requests

\- Reads JSON responses

\- Displays API status codes



\### Run



```bash

python api\_test.py

```
